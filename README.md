## Comprehensive Overview

Orbit is a full-stack decentralized finance protocol built natively on Stellar using Soroban smart contracts. Architected across 6 progressive development phases over ~2 months, Orbit transforms basic yield aggregation into an institutional-grade, multi-product DeFi suite:

### Core Protocol Stack & Features

1. **Soroban Smart Yield Vaults (`orbit-vault`)**
   - **SEP-41 Compliant Share Tokens**: Depositing XLM automatically mints `oXLM` share tokens representing proportional ownership of the vault's Net Asset Value (NAV).
   - **Real Yield via Blend Protocol**: The vault deploys idle capital directly into Blend Protocol lending markets via cross-contract calls, earning real, non-inflationary APY.
   - **Performance & Management Fee Accounting**: Automated fee deductions allocated to governance recipients upon compound harvests.

2. **Yield Tranching Super-Protocol (`orbit-tranche`)**
   - **Principal Tokens (PT)**: Users wrap their `oXLM` vault shares to strip out the underlying base capital into PT tokens. PT holders are guaranteed redemption for their exact initial XLM value, eliminating impermanent loss.
   - **Yield Tokens (YT)**: The remaining variable yield generated by the vault is tokenized into YT tokens. YT holders receive 100% of all future yield generated above the initial principal.

3. **P2P Collateralized Money Market (`orbit-market`)**
   - **Trustless Peer-to-Peer Loans**: Lenders create USDC fixed-term loan offers on-chain. Borrowers lock their Orbit PT or YT tokens as collateral to borrow USDC instantly.
   - **Zero Liquidation Bots Required**: Escrow-based collateral locking enforces fixed duration repayment windows directly in WASM logic without requiring oracle liquidators.

4. **Real-Time On-Chain Indexer & Analytics**
   - **Direct Soroban RPC Ledger Polling**: TVL, deposit volume, historical APY curves, and user rankings are indexed directly from Soroban contract events using custom polling logic without relying on external centralized graph nodes.

5. **User Onboarding & Supabase Auth**
   - Seamless wallet authentication flow syncing user wallet addresses and custom display names to a Supabase backend for global leaderboard personalization.

6. **Gamified Points & Referral Engine**
   - **Orbit Points**: Earn points on every deposit and referral. Share unique referral links to climb global rankings and unlock protocol tier incentives.

7. **Stellar Ecosystem Integrations**
   - **SEP-24 Fiat On-Ramp**: Integrated test anchor modal allowing users to buy crypto with test fiat.
   - **SEP-40 Price Oracles**: Live XLM/USD price feeds converting TVL and portfolio values into USD in real time.
   - **Progressive Web App (PWA)**: Desktop and mobile installable app with offline capabilities and smooth hardware acceleration.

## System Architecture

```
┌─────────────────────────────────────────────────────────────────────┐
│                            USER FLOW                                │
└─────────────────────────────────────────────────────────────────────┘

  User Wallet (Freighter / Albedo / Lobstr / xBull)
        │
        ▼
  ┌─────────────┐     Deposit XLM      ┌──────────────────────────────┐
  │  Orbit App  │ ──────────────────▶  │   orbit-vault (Soroban)      │
  │ (TanStack)  │                      │                              │
  │             │ ◀──────────────────  │  • Mints oXLM share tokens   │
  └─────────────┘   Mint oXLM shares   │  • Tracks TVL & Share NAV    │
        │                              │  • Manages yield strategy    │
        │                              └──────────────┬───────────────┘
        │                                             │
        │                                             │ Yield Strategy
        │                                             ▼
        │                              ┌──────────────────────────────┐
        │                              │     Blend Protocol Pool      │
        │                              │  (Cross-Contract Lending)    │
        │                              └──────────────────────────────┘
        │
        │  YIELD TRANCHING
        │
        ├──── Wrap Shares ───────────▶ ┌──────────────────────────────┐
        │                              │  orbit-tranche (Soroban)     │
        │   ┌─── PT Token (Principal)─ │                              │
        │   │                          │  Splits oXLM into:           │
        │   └─── YT Token (Yield)───── │  • PT: Fixed Principal value │
        │                              │  • YT: Variable Yield claims │
        │                              └──────────────────────────────┘
        │
        │  P2P MONEY MARKET
        │
        └──── Create Offer / Borrow ─▶ ┌──────────────────────────────┐
                                       │  orbit-market (Soroban)      │
                                       │  • Trustless Escrow          │
                                       │  • Fixed Terms & Collateral  │
                                       └──────────────────────────────┘

─────────────────────────────────────────────────────────────────────
                     DATA INFRASTRUCTURE
─────────────────────────────────────────────────────────────────────

  Soroban RPC ──▶ fetchContractEvents() ──▶ Analytics & Leaderboards
  Supabase ──▶ User Profiles & Auth ──▶ Global Leaderboard Names
  SEP-40 Oracle ──▶ XLM/USD Price Feed ──▶ Portfolio Valuation
```

## Key Features

| Component | Functionality |
|-----------|---------------|
| **Multi-Wallet Support** | Integrated via StellarWalletsKit for seamless connection across wallets. |
| **Soroban Smart Vaults** | SEP-41 compliant `oXLM` share token minting with NAV calculations. |
| **Yield Tranching** | Split shares into PT (principal protection) and YT (yield speculation) tokens. |
| **P2P Collateral Market** | Escrow-based lending market utilizing PT/YT tokens as collateral. |

| **Mobile PWA** | Progressive Web App support for mobile and desktop environments. |
| **SEP-40 Price Oracles** | Live USD valuation of vault assets and user share balances. |

## Technical Setup

### Prerequisites

- Node.js 20+
- Rust with `wasm32v1-none` compilation target
- `stellar-cli` v27+

### Quick Start

```bash
# Clone the repository
git clone https://github.com/Sristipriya/stellarorbit.git
cd stellarorbit

# Install dependencies
npm install

# Configure environment
cp .env.example .env

# Run development server
npm run dev
```

## Contract Build Commands

```bash
# Build tranche contract
stellar contract build --package orbit-tranche

# Build vault contract
stellar contract build --package orbit-vault

# Deploy to Testnet
stellar contract deploy \
  --wasm target/wasm32v1-none/release/orbit_vault.wasm \
  --source-account orbit-deployer \
  --network testnet
```
